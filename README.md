# Personal Academic Website

This repository contains the source code for my personal academic website.

本项目为我的个人学术主页源代码，用于展示我的学术经历、研究兴趣、田野调查、发表成果、个人简历以及联系方式等内容。

## Inspiration

The design and structure of this website were inspired by the personal homepage of Professor Wen Xie from the Department of Sociology, Peking University: [https://wen-xie.github.io/](https://wen-xie.github.io/). Her website provided an important reference for me in terms of layout, clarity, and academic presentation.

本网站的设计与结构受到北京大学社会学系谢雯老师个人主页的启发：[https://wen-xie.github.io/](https://wen-xie.github.io/)。谢雯老师主页清晰、简洁而具有学术感的呈现方式，为我搭建自己的个人学术主页提供了重要参考。

## Technical Assistance

The coding work of this website was completed with the assistance of ChatGPT. Since I do not have a formal background in programming, ChatGPT helped me generate, revise, and organize the HTML, CSS, and related code used in this project.

本网站的代码工作是在 ChatGPT 的辅助下完成的。由于我没有系统的编程训练，ChatGPT 帮助我生成、修改和整理了本项目所使用的 HTML、CSS 及相关代码。
